    new PHPQueryTable(rootElement, optionsopt)
    PHPQueryTable.js, line 21

    Initialize the Table with presets and user settings and Set up all button triggers.
    Parameters:
    Name Type Attributes Description
    rootElement HTMLElement The element in which to put the table, preferably a div.
    options Object <optional>
    		 Array of settings that define the behavior of said table.
    Properties
    Name Type Attributes Default Description
    url String <optional>
    		 The url of the associated PHP script for AJAX calls.
    inputClass String <optional>
    		 form-control input-sm The CSS class of the inputs used in the table during edit mode. Default uses Bootstrap.css.
    toolbarClass String <optional>
    		 btn-toolbar The CSS class of the buttons used in the toolbar during when the table is editable.
    groupClass String <optional>
    		 btn-group btn-group-sm The CSS class of the button group used in the toolbar when the table is editable. Default uses Bootstrap.css.
    warningClass String <optional>
    		 warning The CSS class for the warning buttons in the table (I.E. Delete/Confirm). Default uses Bootstrap.css.
    mutedClass String <optional>
    		 text-muted The CSS class for muted text on the table. Default uses Bootstrap.css.
    eventType String <optional>
    		 click The event that is used to trigger the edit mode. If 'dblclick' (for double click), then the Edit buttons will not be shown in the toolbar.
    rowIdentifier String <optional>
    		 id The name of the ID/Primary Key column.
    hideIdentifier boolean <optional>
    		 false Whether to hide the column that contains the ID/PK identified with options.rowIdentifier.
    toolbar boolean <optional>
    		 true Whether to display the toolbar.
    autoFocus boolean <optional>
    		 true Whether to autoFocus when going into edit mode.
    editButton boolean <optional>
    		 true Whether to show the edit button on the toolbar - Overridden as False if options.eventType is 'dblclick'.
    deleteButton boolean <optional>
    		 true Whether to show the delete button on the toolbar .
    saveButton boolean <optional>
    		 true Whether to show the save button on the toolbar.
    restoreButton boolean <optional>
    		 true Whether to show the restore button on the toolbar.
    refreshButton boolean <optional>
    		 false NOT IMPLEMENTED Whether to show the refresh button in the header toolbar.
    rowNumberer boolean <optional>
    		 false Whether to display a row number column on the right side of the data in the table.
    newLine boolean <optional>
    		 false NOT IMPLEMENTED Whether to display the "Add New Record" option as the bottom row of the table.
    buttons Object <optional>
    		 Content, CSS Style Classes, and associated actions of the buttons on the tables.
    formats Object <optional>
    		 Various formatting strings
    Properties
    Name Type Attributes Default Description
    date String <optional>
    		 MM/DD/YYYY Moment.js Formatting string for the Date types
    useDefaultColumnNames boolean <optional>
    		 false Whether to use column names from metadata or from options.
    columns Array.<Object> <optional>
    		 Properties of various columns
    Properties
    Name Type Attributes Description
    Name String <optional>
    		 Override name for the column: either all of the names must be set, or none of them can be set.
    Type String <optional>
    		 Defines how to display a column (select, text, etc.)
    pageLength number <optional>
    		 20 Defines how many records to show per page.
    debug boolean <optional>
    		 Whether to enable verbose debug console output.
